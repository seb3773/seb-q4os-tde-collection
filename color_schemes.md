### ðŸ“¥ Installation

To install a color scheme:
1. Download the desired `.kcsrc` file
2. Copy to one of these locations:
   - **System-wide:** `/opt/trinity/share/apps/tdedisplay/color-schemes/` (requires root)
   - **User-specific:** `~/.trinity/share/apps/tdedisplay/color-schemes/`
3. Apply via: `Control Panel â†’ Appearance & Themes â†’ Colors`
