# Q4OS 6.1 Released with Trinity Desktop Environment

**Author:** Altus Intel  
**Published:** 13 Sep 2025, 6:27 am GMT+0000

---

Published release of the distribution q4os 6.1, based on the Debian code base and featuring KDE Plasma and Trinity desktop environments. Users can switch between both environments on the same system. The distribution is designed to be lightweight on hardware resources and offers a classic desktop design. The loading image sizes are 2 GB for KDE Plasma (download link) and 1.7 GB for Trinity (download link) for x86_64 architecture with plans to create assemblies for ARM64.

The distribution includes various applications like Desktop Profiler for quick software installation and settings, Software Center for additional program installations, and Lookswitcher for changing appearances. An application is provided for installing the distribution in a separate Windows catalog, allowing parallel installation with Windows without needing a separate disk partition.

## What's New in This Release

- Transition to Debian 13.1 package base from Debian 12
- Introduction of KDE Plasma 6.3.6 instead of KDE Plasma 5
- Update of Trinity desktop to version 14.1.5
- Preparation for new releases for I386 systems with the 5.x branch maintained until June 2028

Enthusiasts are developing q4os assemblies with interfaces styled after Windows XP and Windows 10. Check them out here.
