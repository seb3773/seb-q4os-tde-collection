## 📸 <span style="color: #F77F00;">How to Submit Your Setup</span>

<div style="background-color: #FFF8E7; padding: 20px; border-radius: 8px; margin: 15px 0; border-left: 5px solid #F77F00;">

### Submission Guidelines

**What to Include:**

1. **Screenshots** (required)
   - Full desktop screenshot
   - Optional: Close-ups of interesting details
   - Format: PNG or JPG
   - Resolution: At least 1920x1080 recommended

2. **Setup Information** (required)
   - Q4OS version
   - Desktop Environment (Trinity, Plasma, etc.)
   - Theme/Color scheme used
   - Icon set
   - Cursor theme
   - Any custom modifications

3. **Hardware Specs** (optional but appreciated)
   - CPU
   - RAM
   - GPU (if relevant)
   - Age of hardware

4. **Description** (optional)
   - What inspired your setup?
   - Any tips or tricks?
   - Favorite features?

### How to Submit

**Method 1: GitHub Pull Request** (Recommended)
1. Fork this repository
2. Add your screenshots to `assets/user_galleries/your-username/`
3. Add your entry to this file following the template below
4. Submit a pull request

**Method 2: GitHub Issue**
1. Create a new issue with the label "User Gallery Submission"
2. Upload your screenshots
3. Provide the information listed above
4. We'll add it to the gallery for you

**Method 3: Forum/Discord**
- Share on the Q4OS Forum or Discord
- Tag it with #UserGallery or #ShowYourSetup
- We'll reach out to feature it here

</div>

## 💡 <span style="color: #F77F00;">Tips for Great Screenshots</span>

<div style="background-color: #FFF8E7; padding: 20px; border-radius: 8px; margin: 15px 0;">

1. **Clean your desktop** - Close unnecessary windows, organize icons
2. **Show variety** - Include different applications open
3. **Good lighting** - If photographing a screen, ensure good lighting
4. **Native resolution** - Use your monitor's native resolution
5. **Show personality** - Include elements that make your setup unique
6. **Multiple angles** - Desktop, close-ups, system monitor, etc.
7. **Compress images** - Keep file sizes reasonable (use PNG compression)
