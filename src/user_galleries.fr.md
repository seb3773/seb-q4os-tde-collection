## 📋 <span style="color: #2E86AB;">À Propos</span>

<div style="background-color: #F0F8FF; padding: 20px; border-radius: 8px; margin: 15px 0; border-left: 5px solid #2E86AB;">

Cette galerie présente **47 configurations réelles de Q4OS et Trinity Desktop Environment** collectées auprès des membres de la communauté du monde entier. Chaque configuration démontre la flexibilité et les possibilités de personnalisation de Q4OS/TDE.

> **ℹ️ Note sur l'attribution :** Ces captures d'écran ont été collectées à partir de diverses sources, notamment des forums communautaires, des réseaux sociaux (Facebook, etc.) et des soumissions d'utilisateurs au fil du temps. Malheureusement, les informations complètes d'attribution (auteurs, versions OS, détails) n'ont pas toujours été préservées. **Si vous reconnaissez votre configuration ou si vous avez des informations sur une capture, contactez-nous afin que nous puissions ajouter le crédit approprié !**

## 📸 <span style="color: #F77F00;">Comment Soumettre Votre Configuration</span>

<div style="background-color: #FFF8E7; padding: 20px; border-radius: 8px; margin: 15px 0; border-left: 5px solid #F77F00;">

### Directives de Soumission

**À Inclure :**

1. **Captures d'écran** (obligatoire)
   - Capture complète du bureau
   - Optionnel : Gros plans sur des détails intéressants
   - Format : PNG ou JPG
   - Résolution : Au moins 1920x1080 recommandé

2. **Informations sur la Configuration** (obligatoire)
   - Version de Q4OS
   - Environnement de Bureau (Trinity, Plasma, etc.)
   - Thème/Scheme de couleurs utilisé
   - Jeu d'icônes
   - Thème de curseurs
   - Toutes modifications personnalisées

3. **Spécifications Matérielles** (optionnel mais apprécié)
   - CPU
   - RAM
   - GPU (si pertinent)
   - Âge du matériel

4. **Description** (optionnel)
   - Qu'est-ce qui a inspiré votre configuration ?
   - Des astuces ou conseils ?
   - Fonctionnalités préférées ?

### Comment Soumettre

**Méthode 1 : Pull Request GitHub** (Recommandé)
1. Forkez ce dépôt
2. Ajoutez vos captures dans `assets/user_galleries/votre-pseudo/`
3. Ajoutez votre entrée à ce fichier en suivant le modèle ci-dessous
4. Soumettez une pull request

**Méthode 2 : Issue GitHub**
1. Créez une nouvelle issue avec le label "User Gallery Submission"
2. Téléchargez vos captures d'écran
3. Fournissez les informations listées ci-dessus
4. Nous l'ajouterons à la galerie pour vous

**Méthode 3 : Forum/Discord**
- Partagez sur le Forum Q4OS ou Discord
- Taguez avec #UserGallery ou #ShowYourSetup
- Nous vous contacterons pour la présenter ici

</div>

## 💡 <span style="color: #F77F00;">Astuces pour de Belles Captures</span>

<div style="background-color: #FFF8E7; padding: 20px; border-radius: 8px; margin: 15px 0;">

1. **Nettoyez votre bureau** - Fermez les fenêtres inutiles, organisez les icônes
2. **Montrez la variété** - Incluez différentes applications ouvertes
3. **Bon éclairage** - Si photographiez un écran, assurez un bon éclairage
4. **Résolution native** - Utilisez la résolution native de votre moniteur
5. **Montrez votre personnalité** - Incluez des éléments qui rendent votre configuration unique
6. **Angles multiples** - Bureau, gros plans, moniteur système, etc.
7. **Compressez les images** - Gardez des tailles de fichier raisonnables (utilisez la compression PNG)
