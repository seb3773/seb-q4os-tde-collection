## 📋 <span style="color: #2E86AB;">Acerca de</span>

<div style="background-color: #F0F8FF; padding: 20px; border-radius: 8px; margin: 15px 0; border-left: 5px solid #2E86AB;">

Esta galería presenta **47 configuraciones reales de Q4OS y Trinity Desktop Environment** recopiladas de miembros de la comunidad de todo el mundo. Cada configuración demuestra la flexibilidad y las posibilidades de personalización de Q4OS/TDE.

> **ℹ️ Nota sobre la atribución:** Estas capturas de pantalla fueron recopiladas de varias fuentes, incluyendo foros comunitarios, redes sociales (Facebook, etc.) y contribuciones de usuarios a lo largo del tiempo. Desafortunadamente, la información completa de atribución (autores, versiones del SO, detalles) no siempre se conservó. **¡Si reconoce su configuración o tiene información sobre alguna captura, contáctenos para que podamos agregar el crédito correspondiente!**

## 📸 <span style="color: #F77F00;">Cómo Enviar su Configuración</span>

<div style="background-color: #FFF8E7; padding: 20px; border-radius: 8px; margin: 15px 0; border-left: 5px solid #F77F00;">

### Directrices de Envío

**Qué Incluir:**

1. **Capturas de Pantalla** (requerido)
   - Captura completa del escritorio
   - Opcional: Primeros planos de detalles interesantes
   - Formato: PNG o JPG
   - Resolución: Al menos 1920x1080 recomendado

2. **Información de la Configuración** (requerido)
   - Versión de Q4OS
   - Entorno de Escritorio (Trinity, Plasma, etc.)
   - Tema/Esquema de color usado
   - Conjunto de iconos
   - Tema de cursores
   - Cualquier modificación personalizada

3. **Especificaciones de Hardware** (opcional pero apreciado)
   - CPU
   - RAM
   - GPU (si es relevante)
   - Edad del hardware

4. **Descripción** (opcional)
   - ¿Qué inspiró su configuración?
   - ¿Algunos consejos o trucos?
   - ¿Características favoritas?

### Cómo Enviar

**Método 1: Pull Request de GitHub** (Recomendado)
1. Bifurque este repositorio
2. Agregue sus capturas a `assets/user_galleries/su-usuario/`
3. Agregue su entrada a este archivo siguiendo la plantilla abajo
4. Envíe un pull request

**Método 2: Issue de GitHub**
1. Cree un nuevo issue con la etiqueta "User Gallery Submission"
2. Suba sus capturas de pantalla
3. Proporcione la información listada arriba
4. La agregaremos a la galería por usted

**Método 3: Foro/Discord**
- Comparta en el Foro Q4OS o Discord
- Etiquete con #UserGallery o #ShowYourSetup
- Nos pondremos en contacto para presentarla aquí

</div>

## 💡 <span style="color: #F77F00;">Consejos para Grandes Capturas</span>

<div style="background-color: #FFF8E7; padding: 20px; border-radius: 8px; margin: 15px 0;">

1. **Limpie su escritorio** - Cierre ventanas innecesarias, organice iconos
2. **Muestre variedad** - Incluya diferentes aplicaciones abiertas
3. **Buena iluminación** - Si fotografía una pantalla, asegure buena iluminación
4. **Resolución nativa** - Use la resolución nativa de su monitor
5. **Muestre personalidad** - Incluya elementos que hagan única su configuración
6. **Múltiples ángulos** - Escritorio, primeros planos, monitor de sistema, etc.
7. **Comprima imágenes** - Mantenga tamaños de archivo razonables (use compresión PNG)
