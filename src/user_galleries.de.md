## 📸 <span style="color: #F77F00;">Wie reicht man sein Setup ein</span>

<div style="background-color: #FFF8E7; padding: 20px; border-radius: 8px; margin: 15px 0; border-left: 5px solid #F77F00;">

### Einreichungsrichtlinien

**Was einzuschließen ist:**

1. **Screenshots** (erforderlich)
   - Vollständiger Desktop-Screenshot
   - Optional: Nahaufnahmen interessanter Details
   - Format: PNG oder JPG
   - Auflösung: Mindestens 1920x1080 empfohlen

2. **Setup-Informationen** (erforderlich)
   - Q4OS Version
   - Desktop-Umgebung (Trinity, Plasma, etc.)
   - Verwendetes Theme/Farbschema
   - Icon-Set
   - Cursor-Thema
   - Alle benutzerdefinierten Änderungen

3. **Hardware-Spezifikationen** (optional aber geschätzt)
   - CPU
   - RAM
   - GPU (falls relevant)
   - Alter der Hardware

4. **Beschreibung** (optional)
   - Was hat Ihr Setup inspiriert?
   - Tipps oder Tricks?
   - Lieblingsfunktionen?

### Wie man einreicht

**Methode 1: GitHub Pull Request** (Empfohlen)
1. Forken Sie dieses Repository
2. Fügen Sie Ihre Screenshots zu `assets/user_galleries/ihr-benutzername/` hinzu
3. Fügen Sie Ihren Eintrag zu dieser Datei hinzu, indem Sie die Vorlage unten verwenden
4. Reichen Sie einen Pull Request ein

**Methode 2: GitHub Issue**
1. Erstellen Sie ein neues Issue mit dem Label "User Gallery Submission"
2. Laden Sie Ihre Screenshots hoch
3. Geben Sie die oben aufgeführten Informationen an
4. Wir fügen es für Sie zur Galerie hinzu

**Methode 3: Forum/Discord**
- Teilen Sie im Q4OS Forum oder Discord
- Taggen Sie mit #UserGallery oder #ShowYourSetup
- Wir werden uns melden, um es hier zu präsentieren

## 💡 <span style="color: #F77F00;">Tipps für großartige Screenshots</span>

<div style="background-color: #FFF8E7; padding: 20px; border-radius: 8px; margin: 15px 0;">

1. **Reinigen Sie Ihren Desktop** - Schließen Sie unnötige Fenster, organisieren Sie Symbole
2. **Zeigen Sie Vielfalt** - Fügen Sie verschiedene geöffnete Anwendungen ein
3. **Gute Beleuchtung** - Wenn Sie einen Bildschirm fotografieren, sorgen Sie für gute Beleuchtung
4. **Native Auflösung** - Verwenden Sie die native Auflösung Ihres Monitors
5. **Zeigen Sie Persönlichkeit** - Fügen Sie Elemente hinzu, die Ihr Setup einzigartig machen
6. **Mehrere Winkel** - Desktop, Nahaufnahmen, Systemmonitor, etc.
7. **Komprimieren Sie Bilder** - Halten Sie Dateigrößen angemessen (PNG-Kompression verwenden)
