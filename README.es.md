## 🎨 Personalización de UI y Temas
Mejora el atractivo visual de Trinity :-)

### [Windows 10 Window Decoration](https://github.com/seb3773/tde-win-deco-Q4WIN10)
Un motor de decoración de ventanas especializado para TDE diseñado para replicar casi perfectamente el estilo de bordes de Windows 10, proporcionando una interfaz familiar y elegante para los usuarios que se pasan a Linux.

### [Q4WIN10 Application Style](https://github.com/seb3773/tdestyle-Q4WIN10)
Un estilo de widget nativo de TDE que trabaja en tándem con las decoraciones de ventanas para ofrecer una apariencia y sensación de Windows 10 consistente en todas las aplicaciones de Trinity.

### [Q4OS XPack](https://github.com/seb3773/q4osXpack)
Un "Experience Pack" completo que contiene varias mejoras de UI, iconos adicionales y ajustes visuales diseñados para pulir la interfaz de usuario general de Q4OS. (un poco obsoleto, trabajando en una versión moderna)

### [Q4Riv](https://github.com/TheAsmitKid/Q4Riv)
Un tema de Plymouth minimalista y elegante para Q4OS.

### [Trinity Desktop Color Schemes](./src/color_schemes.md)
Una colección de 73 esquemas de color personalizados para Trinity Desktop Environment. Incluye vistas previas visuales e instrucciones de instalación fáciles para personalizar la apariencia de tu escritorio.

### [Dekorator Window Decoration Themes](./src/dekorator_resources.md)
Una colección curada de 56 temas Dekorator para decoraciones de ventanas TDE. Cada tema incluye vistas previas de botones e instrucciones de instalación para personalizar tus bordes de ventanas.

### [Cursors collection](./src/cursors.md)
Varios cursores de ratón diferentes compatibles con TDE.

### [Icons sets](./src/icons.md)
Varios temas de iconos.

### [Plymouth boot themes collection](./src/boot_plymouth.md)
Temas de arranque Plymouth.

### [KColorSchema](https://github.com/seb3773/kcolorschema)
Un convertidor rápido de '.colorscheme' a '.schema' para Konsole, permitiendo una integración perfecta de esquemas de color en la terminal de Trinity.

### [Konsole .schema themes collection](https://github.com/seb3773/konsole-schemas-collection)
Una colección curada de temas .schema para Trinity Konsole.

### [TDE panel skins](https://github.com/splatert/tdepanelskins)
Skins de barra de tareas/panel para Trinity Desktop Environment.

### [Fonts](./src/fonts.md)
Muchas fuentes :-)

### [Trinity Look](https://www.trinity-look.org/)
Adornos para tu escritorio Trinity.

## 🛠️ Herramientas de Escritorio y Applets


### [Modern KMenu: classic-x](https://github.com/seb3773/tde-kmenu_classic-x)
Una actualización para el menú clásico de TDE. Integra una búsqueda instantánea al estilo Windows 10, refinamientos de la barra lateral y otras mejoras.

### [Show Desktop Applet](https://github.com/seb3773/showdeskten-kicker-applet)
Un applet "Mostrar escritorio" al estilo Windows 10 para el panel Kicker. Proporciona una manera conveniente de minimizar todas las ventanas y echar un vistazo a tu escritorio con un solo clic.

### [QSidebar](https://github.com/seb3773/qsidebar)
Una utilidad de barra lateral ligera y versátil para Q4OS. Ofrece acceso rápido a información del sistema, aplicaciones utilizadas frecuentemente y atajos personalizados. Está basado en Gtk3, pero ofrece integración completa con el entorno Trinity Desktop a través de un applet Kicker dedicado.

### [QSuperL](https://github.com/seb3773/qsuperl)
Una útil utilidad para gestionar el comportamiento de la tecla "Super" (Windows). Te permite mapear la tecla para abrir el menú de aplicaciones o activar acciones personalizadas del sistema.

### [Xakar](https://github.com/TheAsmitKid/Xakar)
Daemon de mosaico X11 nativo en C++.

## 🚀 Sistema y Productividad
Scripts de optimización y utilidades para mantener tu sistema funcionando sin problemas.

### [Q4OS XPack (System Tweaks)](https://github.com/seb3773/q4osXpack)
Más allá de los cambios de UI, el XPack incluye una colección de optimizaciones a nivel de sistema y configuraciones de software adicionales para mejorar el rendimiento y la usabilidad de Q4OS.

### [Compton TDE optimized](https://github.com/seb3773/compton-tde-x)
Una versión optimizada del compositor Compton (X11). Orientado al rendimiento y bajo uso de recursos.

### [Konsole Little Theme Manager](https://github.com/seb3773/Konsole-little-theme-manager)
Una herramienta basada en CLI diseñada para gestionar y cambiar fácilmente entre diferentes temas de color para la terminal de Trinity (Konsole).

### [Leafbar](https://github.com/blu256/leafbar)
Leafbar es un panel para TDE en el espíritu de Deskbar, el panel de escritorio de BeOS y ahora Haiku.

### [TDE Easy Projector](https://github.com/blu256/tde-easy-projector)
Utilidad extremadamente simple para configurar un proyector para una presentación.

### [Q4OS i18n](https://github.com/seb3773/q4os-i18n)
Un repositorio dedicado a la internacionalización de Q4OS. Contiene archivos de traducción y scripts de localización para hacer el escritorio accesible en múltiples idiomas, ¡¡no dudes en contribuir!!

## 🖼️ Fondos de Pantalla y Multimedia
Herramientas automatizadas para mantener tus fondos de escritorio frescos y dinámicos.

### [Q4OS & Trinity dedicated wallpapers](./src/wallpapers.md)
Una hermosa colección de fondos de pantalla diseñados específicamente para Q4OS y Trinity Desktop Environment. Presenta varios estilos artísticos incluyendo renderizados 3D, efectos de vidrio y diseños de la mascota Konqi.

### [MS Theme Pack Installer](https://github.com/seb3773/msthemepack_installer)
Un script inteligente que te permite instalar y extraer archivos `.themepack` oficiales de Microsoft en Linux. Establece automáticamente los fondos de pantalla incluidos e intenta coincidir con los colores del sistema.

### [NASA Wallpapers for TDE](https://github.com/seb3773/nasa-wallpapers-for-linux_tde)
Trae las maravillas del espacio a tu escritorio. Este script obtiene la "Imagen Astronómica del Día" de la NASA y la establece como tu fondo de pantalla TDE automáticamente.

### [Bing Wallpapers for TDE](https://github.com/seb3773/bing-wallpapers-for-linux_tde)
Mantente inspirado con imágenes diarias de alta calidad. Esta utilidad sincroniza tu fondo de escritorio con la imagen actual del día de la página principal de Bing.

### [KMenu Start Button Images](./src/kmenu_start_img.md)
Una colección completa de imágenes personalizadas de botón de inicio para Trinity KMenu. Más de 90 diseños diferentes incluyendo temas Q4OS, estilos Windows, iconos informáticos retro y diseños modernos creativos para personalizar tu barra de tareas.

## 📖 DOCUMENTACIÓN / CÓMO HACER
Varios documentación, recursos, consejos, etc...

### [Q4OS FAQ - Trinity Desktop Manual](./src/Q4OS_FAQ.es.md)
Una guía FAQ completa para Q4OS con Trinity Desktop Environment del Equipo Q4OS. Cubre instalación, configuración del sistema, personalización del escritorio y solución de problemas.

### [Q4OS Setup on Debian](./src/Q4OS_Setup_on_Debian.md)
Guía del administrador para instalar Q4OS sobre cualquier distribución basada en Debian (Debian, Devuan, Armbian, Raspberry Pi OS). Incluye soporte ARM64.

### [Q4OS Setup and Using Guide](./src/Q4OS_Setup_and_Using.es.md)
Manual de usuario completo que cubre la configuración, instalación y uso diario de Q4OS. Incluye configuración de hardware, impresión, gestión de energía, localización y consejos del sistema.

### [Q4OS Desktop Profiler Guide](./src/Q4OS_Desktop_Profiler.md)
Manual de usuario para la herramienta Desktop Profiler. Aprende a aplicar perfiles, crear perfiles personalizados y automatizar la instalación de Q4OS con conjuntos de aplicaciones preconfigurados.

### [Q4OS Custom Application Installer](./src/Q4OS_Custom_Installer.md)
Guía del desarrollador para crear instaladores personalizados de Q4OS (paquetes .qsi y .deb). Incluye ejemplos para Apt, Flatpak y plantillas avanzadas de instalador.

### [Build an Application for Trinity Desktop](./src/Q4OS_Build_Trinity_App.md)
Manual del desarrollador con recomendaciones generales para crear aplicaciones para Q4OS Trinity. Cubre reglas de instalación, estructura de directorios, empaquetado con Apt y directrices de portabilidad.
