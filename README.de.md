## 🎨 UI-Anpassung & Themen
Verbessern Sie die visuelle Attraktivität von Trinity :-)

### [Windows 10 Window Decoration](https://github.com/seb3773/tde-win-deco-Q4WIN10)
Eine spezialisierte Fensterdekoration für TDE, die entwickelt wurde, um den Windows 10-Rahmenstil fast perfekt zu replizieren und eine vertraute und elegante Benutzeroberfläche für Linux-Umsteiger bietet.

### [Q4WIN10 Application Style](https://github.com/seb3773/tdestyle-Q4WIN10)
Ein nativer TDE-Widget-Stil, der mit den Fensterdekorationen zusammenarbeitet, um ein konsistentes Windows 10-Look-and-Feel in allen Trinity-Anwendungen zu bieten.

### [Q4OS XPack](https://github.com/seb3773/q4osXpack)
Ein umfassendes "Experience Pack" mit verschiedenen UI-Verbesserungen, zusätzlichen Icons und visuellen Anpassungen, die darauf abzielen, die Q4OS-Benutzeroberfläche zu verfeinern. (etwas veraltet, eine moderne Version ist in Arbeit)

### [Q4Riv](https://github.com/TheAsmitKid/Q4Riv)
Ein minimales, elegantes Plymouth-Thema für Q4OS.

### [Trinity Desktop Color Schemes](./src/color_schemes.md)
Eine Sammlung von 73 benutzerdefinierten Farbschemata für Trinity Desktop Environment. Enthält visuelle Vorschauen und einfache Installationsanleitungen zur Anpassung Ihres Desktop-Aussehens.

### [Dekorator Window Decoration Themes](./src/dekorator_resources.md)
Eine kuratierte Sammlung von 56 Dekorator-Themen für TDE-Fensterdekorationen. Jedes Thema enthält Button-Vorschauen und Installationsanleitungen zur Personalisierung Ihrer Fensterrahmen.

### [Cursors collection](./src/cursors.md)
Verschiedene Mauszeiger, die mit TDE kompatibel sind.

### [Icons sets](./src/icons.md)
Verschiedene Icon-Themen.

### [Plymouth boot themes collection](./src/boot_plymouth.md)
Plymouth-Boot-Themen.

### [KColorSchema](https://github.com/seb3773/kcolorschema)
Ein schneller '.colorscheme' zu '.schema' Konverter für Konsole, der eine nahtlose Integration von Farbschemata im Trinity-Terminal ermöglicht.

### [Konsole .schema themes collection](https://github.com/seb3773/konsole-schemas-collection)
Eine kuratierte Sammlung von .schema-Themen für Trinity Konsole.

### [TDE panel skins](https://github.com/splatert/tdepanelskins)
Taskleisten/Panel-Skins für Trinity Desktop Environment.

### [Fonts](./src/fonts.md)
Viele Schriftarten :-)

### [Trinity Look](https://www.trinity-look.org/)
Augenschmaus für Ihren Trinity Desktop.

## 🛠️ Desktop-Tools & Applets


### [Modern KMenu: classic-x](https://github.com/seb3773/tde-kmenu_classic-x)
Ein Upgrade für das klassische TDE-Menü. Es integriert eine Windows 10-artige Sofort-Suche, Verfeinerungen der Seitenleiste und andere Verbesserungen.

### [Show Desktop Applet](https://github.com/seb3773/showdeskten-kicker-applet)
Ein "Desktop anzeigen" Applet im Windows 10-Stil für das Kicker-Panel. Es bietet eine praktische Möglichkeit, alle Fenster zu minimieren und mit einem einzigen Klick auf Ihren Desktop zu schauen.

### [QSidebar](https://github.com/seb3773/qsidebar)
Ein leichtes und vielseitiges Seitenleisten-Dienstprogramm für Q4OS. Es bietet schnellen Zugriff auf Systeminformationen, häufig verwendete Anwendungen und benutzerdefinierte Verknüpfungen. Es basiert auf Gtk3, bietet jedoch eine vollständige Integration in die Trinity Desktop Environment durch ein dediziertes Kicker-Applet.

### [QSuperL](https://github.com/seb3773/qsuperl)
Ein praktisches Dienstprogramm zur Verwaltung des Verhaltens der "Super"-Taste (Windows-Taste). Es ermöglicht Ihnen, die Taste dem Öffnen des Anwendungsmenüs oder dem Auslösen benutzerdefinierter Systemaktionen zuzuordnen.

### [Xakar](https://github.com/TheAsmitKid/Xakar)
Nativer C++ X11 Tiling Daemon.

## 🚀 System & Produktivität
Optimierungsskripte und Dienstprogramme, um Ihr System reibungslos am Laufen zu halten.

### [Q4OS XPack (System Tweaks)](https://github.com/seb3773/q4osXpack)
Über UI-Änderungen hinaus enthält das XPack eine Sammlung von Optimierungen auf Systemebene und zusätzliche Softwarekonfigurationen zur Verbesserung der Leistung und Benutzerfreundlichkeit von Q4OS.

### [Compton TDE optimized](https://github.com/seb3773/compton-tde-x)
Eine optimierte Version des Compton (X11) Compositors. Zielgerichtet auf Leistung und geringen Ressourcenverbrauch.

### [Konsole Little Theme Manager](https://github.com/seb3773/Konsole-little-theme-manager)
Ein CLI-basiertes Tool, das entwickelt wurde, um verschiedene Farbthemen für das Trinity-Terminal (Konsole) einfach zu verwalten und zu wechseln.

### [Leafbar](https://github.com/blu256/leafbar)
Leafbar ist ein Panel für TDE im Geiste von Deskbar, dem Desktop-Panel von BeOS und jetzt Haiku.

### [TDE Easy Projector](https://github.com/blu256/tde-easy-projector)
Extrem einfaches Dienstprogramm zum Einrichten eines Projektors für eine Präsentation.

### [Q4OS i18n](https://github.com/seb3773/q4os-i18n)
Ein Repository, das der Internationalisierung von Q4OS gewidmet ist. Es enthält Übersetzungsdateien und Lokalisierungsskripte, um den Desktop in mehreren Sprachen zugänglich zu machen. Zögern Sie nicht, beizutragen !!

## 🖼️ Hintergrundbilder & Medien
Automatisierte Tools, um Ihre Desktop-Hintergründe frisch und dynamisch zu halten.

### [Q4OS & Trinity dedicated wallpapers](./src/wallpapers.md)
Eine schöne Sammlung von Hintergrundbildern, die speziell für Q4OS und Trinity Desktop Environment entwickelt wurden. Enthält verschiedene künstlerische Stile einschließlich 3D-Renderings, Glaseffekten und Konqi-Maskottchen-Designs.

### [MS Theme Pack Installer](https://github.com/seb3773/msthemepack_installer)
Ein cleveres Skript, mit dem Sie offizielle Microsoft `.themepack`-Dateien unter Linux installieren und extrahieren können. Es stellt automatisch die enthaltenen Hintergrundbilder ein und versucht, die Systemfarben anzupassen.

### [NASA Wallpapers for TDE](https://github.com/seb3773/nasa-wallpapers-for-linux_tde)
Bringen Sie die Wunder des Weltraums auf Ihren Desktop. Dieses Skript ruft das NASA "Astronomy Picture of the Day" ab und stellt es automatisch als Ihr TDE-Hintergrundbild ein.

### [Bing Wallpapers for TDE](https://github.com/seb3773/bing-wallpapers-for-linux_tde)
Bleiben Sie inspiriert mit hochwertigen täglichen Bildern. Dieses Dienstprogramm synchronisiert Ihren Desktop-Hintergrund mit dem aktuellen Bing Homepage-Bild des Tages.

### [KMenu Start Button Images](./src/kmenu_start_img.md)
Eine umfassende Sammlung benutzerdefinierter Startbutton-Bilder für das Trinity KMenu. Über 90 verschiedene Designs einschließlich Q4OS-Themen, Windows-Stilen, Retro-Computing-Icons und kreativen modernen Designs zur Personalisierung Ihrer Taskleiste.
